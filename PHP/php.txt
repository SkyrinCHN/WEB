PHP
    1,背景:
        1994年诞生,为了记录网页访问情况;
        1999年改名.
    2,特点:开源.简单.跨平台.占用资源少,尤其适合中小型需要快速开发的WEB项目
     技术栈:LAMP=linux+Apache+Mysql+PHP
     3,PHP 环境搭建
        1,服务器端:安装一款WEB静态服务器,如Apache
        2,服务器端:安装PHP解释器
        3,服务器端:整合配置WEB服务器和PHP解释器
        注意:上面三步可以使用服务器端套件来完成,如XAMFF
        4,服务器端:启动WEB服务器
        5,服务器端:编写网页保存在客户端可以访问到的目录下
            默认路径 C:/XAMFF/htdocs
        6,客户端:打开浏览器,在地址栏中时输入WEB服务器的地址

    3,字符
        1,<br> 换行 <hr>水平分割线 &nbsp 空格 都是可以被浏览器识别的特殊字符
    4变量和常量
        变量:值可能发生改变的数据,如用户的年龄,商品的价格,用户的密码帖子的回复数量等.
            定义一个变量:$变量名=值;
                        $x=2; $y=3;Z=$X+$y;
            输出一个变量:echo $变量名;
            变量名中可以包含数字,字母,下划线,不能以数字开头.
            合法的变量名:$i,$user_age,$user_Age 
            非法的变量名:age,$2age,
    启动WEB服务器
        静态:内容不变,html/css/js/图片..
        动态:内容不同,jsp/asp.net/php/node.js..

    变量和常量:
    $price =3000;
    $count=2;
    程序中变量(Variable)的本质:内存中一块储存空间的别名,变量值的修改就是替换之前
                    在此块内存中的已有数据.
        常量(Constant): 值不能发生改变的数据.
        声明一个常量:        const 常量名 =值;
        输出一个常量:        echo  常量名;
        常量名不用$开头;可以包含数字,字母,下划线,不能以数字开头.习惯上常量名用纯
                    大写形式
                const PI = 3.1415926;
                const WITHDRAW_LIMIT = 20000;
